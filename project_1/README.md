# Brendan Vogel - Personal Portfolio Website

## Purpose
This multi-page website showcases my background, resume, hobbies, and software projects.

## Key Features
- Multi-page navigation with linked pages:
  - `index.html`
  - `channels/project_resume.html`
  - `channels/project_hobbies.html`
  - `channels/project_about.html`
- Semantic HTML elements used across the site:
  - `header`, `main`, `footer`, `nav`, `article`, and `section`
- Custom CSS styling for:
  - Color scheme, typography, borders, spacing, and card-style sections
- Responsive design:
  - Flexbox layouts for page sections and project/hobby content
  - Media queries for mobile-friendly stacking and sizing
- Multimedia and images:
  - Embedded video (`images/RC.mp4`) on project pages
  - Project and hobby images throughout the site

## Project Content Summary
- **Home page** showing a picture of myself as wellas introducing the navigation to the other pages
- **Resume page** with education, experience, skills, courses, and project history
- **Hobbies page** with favorite games, chess profile and ratings, and basketball section
- **About page** with personal background and leadership experience

## File Structure
- `index.html` (homepage)
- `channels/` (subpages)
- `images/` (media assets)
